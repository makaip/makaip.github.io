Projects

CalcLynx

A stack-based infinite LaTeX math whiteboard that lets you create, edit, and manage mathematical expressions using MathQuill and MathGene.

5-HT Freezing

Developing a deep learning model that predicts freezing behavior (fear response) from 5-HT recordings in the orbitofrontal cortex using a 1D CNN backbone with BiLSTM. The model is trained on fiber photometry recordings and computer-scored freezing data, which I am working with at my research internship for the Max Planck Florida Institute for Neuroscience.

CDOM Detection

Working at the Harbor Branch Oceanographic Institute to develop a ML approach for identifying surface expressions of upwelling in the Northern Gulf of Mexico using RS data. We can then use RS-derived estimates of CDOM to assess how these upwelling events may influence surface DOC dynamics.

Mathematix

A nodeblock-based math webapp designed to make graphing calculators more accessible and easy to use. Using node blocks, the interface allows users to drag and drop mathematical operations, enabling them to connect them together in order to calculate certain functions.

The project won first place in the YCW QBitX Hacks Hackathon among over 590 participants nationwide. I also animated a short advertisment in Blender that you can view here.

Clevulator

Written in C# and inspired by Goat Simulator, the game was created in Unity to simulate the experience of being Mr. Clevenger (my former history teacher). Characters were animated using motion capture data from Adobe Mixamo, and Unity Navmesh allows NPCs to walk, talk, and interact with their environment. The environment was created using reality capture software Scaniverse to create accurately-scaled models, which were cleaned up and merged in Blender.

Bakify

An online web application originally created for Ms. Baker's class designed to help users improve their writing by identifying and replacing analytical never-words. The application automatically switches to dark mode at night for optimal user experience, and was also my first web dev project.

Opentest

An online test-taking app designed to look just like the PSAT/SAT using simple Javascript, HTML, and CSS. You can upload custom quizzes and tests in CSV format and set a timer for a set time or unlimited time. You can also pause the test and resume at any time.

Where2Meet

Co-developed full stack webapp inspired by When2Meet, except for location instead of time.

Urinal Splashback Calculator

Vibe-coded out of curiousity to see how the angle of stream intersection affects splashback force.
