Projects

Mathematix

Lead developer and creator of Mathematix, a nodeblock-based math webapp designed to make graphing calculators more accessible and easy to use. Using node blocks, the interface allows users to drag and drop mathematical operations, enabling them to connect them together in order to calculate certain functions. It is designed to be ultra-lightweight, with the entire system created from scratch using JavaScript canvases. The project won first place in the YCW QBitX Hacks Hackathon among over 590 participants nationwide. I also animated a short advertisment in Blender that you can view here.

Clevulator

Written in C# and inspired by Goat Simulator, the game was created in the Unity Game Engine to simulate the experience of Mr. Clevenger (my former history teacher). The player can play as Mr. Clevenger using state-of-the-art motion tracking technology made available by Adobe Mixamo. Furthermore, Unity Navmesh allow for more realistic NPCs, which are able to walk, talk, and interact with their environment. The environment was created using reality capture software Scaniverse to create accurately-scaled models.

Bakify

Bakify, an online web application originally created for Ms. Baker's class, is designed to help users improve their writing by identifying and replacing analytical never-words. By highlighting these weak words that can obstruct your message, Bakify potentially enhances your writing. Additionally, the application automatically switches to dark mode at night for optimal user experience.

Helped with UX/UI design and light frontend development in Alex Castronovo's Multiplayer Sudoku, which you can play online and check out on his website. I also made original icons for the game in Blender. Alex did all the hard stuff I just made it look pretty lol.
